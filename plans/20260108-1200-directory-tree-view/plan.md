## Success Criteria

- Two-pane layout with resizable splitter
- Tree shows directory hierarchy
- Clicking directory in tree updates file list
- Tree syncs with current directory
- Directories can be expanded/collapsed
- New folders appear in tree automatically
- Context menu for tree operations
- Compile succeeds without errors
